Theme Overview

The eye is an efficient 404 responsive Page. Clean, classy and elegant for every creative people, corporates, agency, e-commerce, business, portfolio etc…

Summary

  • Compatible Browsers: IE11, Firefox, Safari, Opera, Chrome, Edge
  • Files Included: HTML Files, CSS Files, JS Files

Icon Usage

Font based icon set FontAwesome are used in this template. Hence the size and color of the icons can be controlled through css. View here the full list of icons used.

    <i class="fa fa-google-plus"></i>
    <i class="fa fa-twitter"></i>
    <i class="fa fa-facebook"></i>
    <i class="fa fa-behance"></i>
    <i class="fa fa-angle-down"></i>
    <i class="fa fa-copyright"></i>
    <i class="fa fa-dribble"></i>
    <i class="fa fa-plus"></i>
    <i class="fa fa-minus"></i>

Icons are connected with the following code into the <head> section:

    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css">

Javascript

  • jquery.fullPage.js A simple and easy to use plugin to create fullscreen scrolling websites.
    It allows the creation of fullscreen scrolling websites, as well as adding some landscape sliders inside the sections of the site.
  • jQuery jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
  • main.js Contents javascripts code to initialize other js plugins present in vendor.js and plugins.js
